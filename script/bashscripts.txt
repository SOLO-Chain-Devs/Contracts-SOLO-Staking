For more expensive gas transactions, you can:

Add data to make the transaction more complex:
cast send --private-key 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80 --value 0.01ether 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266 --data 0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef --rpc-url http://127.0.0.1:8545

Set a higher gas limit explicitly:

cast send --private-key 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80 --value 0.01ether 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266 --gas-limit 100000 --rpc-url http://127.0.0.1:8545

Increase data size proportionally to make transactions use different amounts of gas:

# Wallet 0 (smallest)
cast send --private-key 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80 --value 0.01ether 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266 --data 0x1234 --rpc-url http://127.0.0.1:8545
